
# MooseDB Manager

A custom database manager for easy user experience, with saving, loading, data types, and more.

MooseDB Manager is a database manager designed to be used in conjunction with small businesses/projects that require a quick and easy database management software. I made this project for one of my final projects for SDEV200 at IvyTech.

![General main table with orders, clients, and more](https://github.com/Daryn250/MooseDB-Manager/blob/master/images/general.png)

Features:

- saving
- loading
- adding tables
- removing tables
- data types
    - String data type
    - Integer data type
    - Double data type
    - Paragraph data type
 
![Loading from a .json file](https://github.com/Daryn250/MooseDB-Manager/blob/master/images/loading.png)

Data is saved to a .json file of your choice, and can be easily shared. I used Google's GSON library to assist with saving/loading, as well as a custom function to serialize/deserialize the datatype classes.
